\# AI-Powered Financial Stability Analyzer



An AI-driven data analytics web application that evaluates personal financial stability using income and expense data. The app calculates key financial metrics, visualizes spending trends, and uses Generative AI to produce clear, human-readable financial insights.



---



\## Project Overview



Managing personal finances can be difficult without clear insights.  

This project helps users understand their \*\*financial health\*\* by analyzing income and expenses, calculating savings metrics, and explaining results in simple language using Generative AI.



The application is built as an interactive \*\*Streamlit web app\*\*.



---



\## Key Features



\- Income, expense, and savings analysis  

\- Financial health classification (Stable / Moderate / Risk)  

\- Interactive charts and KPIs  

\- Category-wise expense breakdown  

\- AI-generated financial insights using Generative AI
